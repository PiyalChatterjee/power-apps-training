## ï¿½ğŸš€ Getting Started

### **Prerequisites**

- Microsoft 365 account (Office 365 E3/E5 or Power Platform licenses)
- Power Apps Developer Plan (free) - [Get it here](https://powerapps.microsoft.com/en-us/developerplan/)
- Basic understanding of business processes
- Willingness to learn and practice hands-on

### **Environment Setup**

1. **Sign up for Power Platform Developer Environment**

   ```
   1. Go to https://powerapps.microsoft.com/developerplan/
   2. Sign up with your Microsoft account
   3. Create your developer environment
   4. Verify access to all Power Platform services
   ```

2. **Install Recommended Tools**

   ```
   - Power Platform CLI
   - Visual Studio Code
   - Power Platform Tools extension
   - Git for version control
   ```

3. **Access Training Materials**
   ```bash
   git clone https://github.com/PiyalChatterjee/power-apps-training.git
   cd power-apps-training
   ```

### **Study Approach**

1. **Sequential Learning**: Start with Stage 1 and progress through each stage
2. **Hands-On Practice**: Complete all practical exercises
3. **Build Along**: Create the leave management system components
4. **Review and Reinforce**: Use knowledge checks and assessments
5. **Apply Knowledge**: Implement in your own projects
